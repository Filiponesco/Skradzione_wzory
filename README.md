 - 1: 23,72762
 - 2: 17,77639
 - 3: 12,04160
 - 4: 16,34014
 - 5: 16,58312
 - 6: 16,64332
 - 7: 17,34935
 - 8: 44,65423 <-- max
 - 9: 11,35783
 - 10: 15,19868

Wartość maksymalna została przypisana do zmiennej "maxOfEuclidean" w klasie: "Algorytm.cs"
